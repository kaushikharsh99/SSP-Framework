# Security Policy

## Supported Versions

Only the latest release version on the `main` branch is supported with security updates.

## Reporting a Vulnerability

We take the security of the SSP Framework seriously. If you find any security vulnerabilities or issues, please do not report them publicly via Github issues.

Instead, please send an email to the project maintainers with a description of the vulnerability and steps to reproduce or exploit it.

We will acknowledge your report within 48 hours and work with you to patch the issue before making it public.
