# Installation

This page details the installation guidelines for the SSP Framework.

## Requirements
- Python >= 3.10
- PyTorch >= 2.2.0 (CUDA support is highly recommended)

## Installation from Source

1. Clone the repository:
   ```bash
   git clone https://github.com/kaushikharsh99/SSP-Framework.git
   cd SSP-Framework
   ```

2. Install in editable mode:
   ```bash
   pip install -e .
   ```

## Developer Mode installation
For installing extra dependencies needed for development, testing, and building docs:
```bash
pip install -e ".[dev]"
```
